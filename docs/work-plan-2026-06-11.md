## 3. 潜在问题排查

| 场景 | 检查方法 |
|------|---------|
| bot 启动后状态是 STOPPING/STOPPED | 查看 agent 日志，可能是账号登录失败（网络/账号状态） |
| 命令返回 error | 确认 bot 确实在线；bot 登录需要时间，等 `login_timeout` 后重试 |
| WebSocket 连不上 | 确认 agent 已启动；检查防火墙/端口 |
| 导入失败 | 检查 CSV 数据是否恰好 6 个字段 |
